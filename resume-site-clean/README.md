# Chirag Mehta – Professional Resume & Portfolio

A modern, interactive resume website built with HTML, CSS, and JavaScript.

## Features
- Interactive navigation with smooth transitions
- Responsive design
- Professional layout with neon cyan theme
- Sections for About, Skills, Experience, Projects, Education, and Contact

## How to Share

### Option 1: GitHub Pages (Recommended - Free & Easy)
1. Sign up at [github.com](https://github.com)
2. Create a new repository named `resume-site`
3. Upload these files (index.html, style.css, README.md)
4. Go to Settings → Pages → Select "main" branch
5. Your resume will be live at: `https://username.github.io/resume-site`
6. Share this link with HR!

### Option 2: Netlify (Also Free)
1. Go to [netlify.com](https://netlify.com)
2. Drag and drop this folder
3. Get an instant public link

### Option 3: Local Sharing
Simply open `index.html` in a web browser to view locally.
